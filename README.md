# Insider Threat Detection System (System Monitoring)

## Project Description
This is a Python-based system monitoring project designed to monitor the complete system activity.  
The application tracks system usage and records the **runtime and downtime of specific software** installed on the system.  
It helps in understanding user behavior and identifying unusual or suspicious activities based on software usage patterns.

The project provides an interactive interface using **Streamlit** and stores monitoring data for further analysis.

---

## Features
- Monitors full system activity
- Tracks software runtime and downtime
- Records system usage data
- User-friendly dashboard using Streamlit
- Export and manage data using Excel files

---

## Technologies Used
- Python
- System Monitoring Libraries
- Streamlit (for UI)
- Excel-based data storage
